## 🗂️ Project Overview

This model classifies garbage into 6 categories:
- Cardboard
- Glass
- Metal
- Paper
- Plastic
- Trash

Using TensorFlow and transfer learning, EfficientNetV2B2 was applied to train on an image dataset with strong validation performance.
